# 🚀 Antigravity - Jogo Atari

Um jogo arcade 2D estilo Atari desenvolvido em Python com Pygame.

## 🎮 Como Jogar

- **← →** — Mover a nave para a esquerda/direita
- **Espaço** — Atirar projéteis
- **Enter** — Reiniciar após Game Over

## 🎯 Objetivo

Destrua os asteroides antes que eles cheguem ao fundo da tela ou colidam com a sua nave. Cada asteroide destruído vale **10 pontos**.

## 🛠️ Pré-requisitos

- Python 3.x
- Pygame

```bash
pip install pygame
```

## ▶️ Executar

```bash
cd jogo-atari
python game.py
```
